# Selfhost Fase 3 - Smoke

Dette dokumentet definerer den første minimumssmoken for fase 3.

## Smoken skal dekke

- AST-kontrakt v1 kan lesast og validerast
- Semantic-kjerne v1 kan byggje state og scope
- IR-til-bytecode v1 er dokumentert og klar som normal løype

## Minimumsforventing

- AST-status kan lesast utan å tolke intern implementasjon
- Semantic-status kan rapportere scope og symbolreglar
- IR-status kan vise at bytecode-løypa er normal veg
