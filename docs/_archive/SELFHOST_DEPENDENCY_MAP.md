# Selfhost Dependency Map

Historisk oversikt over den første selfhost-kartlegginga.

## Kva det dekte

- kor dei gamle compiler-/runtime-spora låg
- kva som hadde selfhost-ekvivalent
- kva som framleis var historisk bootstrap/legacy

## Kva det betyr no

- normal CLI er flytta til `./bin/nc`
- selfhost-vegen er normalflata
- dette dokumentet er berre historikk
