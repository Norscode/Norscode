# Full Independence Status

Dette er den historiske sluttstatusen for arbeidet med å skilje normal drift frå legacy-/maintainer-beslag.

## Status

- Normal bruk er rydda til native-first og selfhost-basert flyt.
- README, ROADMAP og dei viktigaste statusdokumenta omtalar no normalvegen som standard.
- Workflow-filene er historisk / opt-in merkte for vedlikehald, ikkje normal drift.
- `tools/maint/*` og `tools/build_norscode_native.sh` er no tydeleg dokumenterte som eksplisitt vedlikehaldsløype.
- Det som står att er den bevisste historiske vedlikehaldsflata og arkivert historikk.
