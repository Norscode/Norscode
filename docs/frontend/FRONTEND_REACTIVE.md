# Frontend Reactive

`std.reactive` gir en liten HTML-kontrakt for interaktive widgets.
Pakken lager `data-*` attributter og små server-renderte elementer som et script kan koble seg på.

## Bindinger

- `bind_text(...)`
- `bind_value(...)`
- `bind_checked(...)`
- `bind_class(...)`
- `bind_visible(...)`
- `bind_attr(...)`
- `state_attr(...)`

## Events

- `click_attr(...)`
- `input_attr(...)`
- `change_attr(...)`
- `submit_attr(...)`
- `focus_attr(...)`
- `blur_attr(...)`
- `keydown_attr(...)`

## UI-Hjelpere

- `reactive_root(...)`
- `reactive_view(...)`
- `bind_panel(...)`
- `action_button(...)`
- `reactive_status(...)`
- `target_attr(...)`
- `busy_attr(...)`
