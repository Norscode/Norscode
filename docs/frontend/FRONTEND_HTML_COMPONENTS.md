# Frontend HTML Components

`std.html_components` er et lite komponentlag oppå `std.html`.
Pakken gir server-renderte byggeklosser som kan brukes alene eller sammen med `std.frontend`.

## Grunnkomponenter

- `knappelenke(...)`
- `knappelenke_sekundær(...)`
- `kort(...)`
- `kort_med_footer(...)`
- `hero(...)`
- `side_topp(...)`
- `side_footer(...)`
- `nav_lenke(...)`

## Innhold Og Status

- `seksjon(...)`
- `funksjon_grid(...)`
- `funksjon_kort(...)`
- `status_badge(...)`
- `callout(...)`
- `callout_info(...)`
- `meta_rad(...)`
- `metrikk_kort(...)`
- `kort_liste(...)`
- `kort_liste_rad(...)`
- `tom_tilstand(...)`
