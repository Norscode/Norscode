# Selfhost Fase 3 - IR-smoke

Dette dokumentet definerer minimumssmoken for IR-til-bytecode-løypa.

## Smoken skal verifisere

- IR-løypa er dokumentert som normal veg
- Bytecode-emisjon er knytt til NCB JSON
- Løypa er klar for vidare utviding utan å bryte normal kontrakt
