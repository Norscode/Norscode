# Selfhost Fase 3 - IR-regresjon

Dette dokumentet samlar den første regresjonsreferansen for IR-løypa.

## Regresjonsmål

- IR til bytecode skal halde same dokumenterte hovudveg
- Endringar i bytecode-emisjon skal vere eksplisitte
- NCB JSON skal framleis vere normal outputflate

## Kva som skal vakast

- `selfhost/compiler/ir_to_bytecode.no`
- eventuell vidare kopling mot semantic-resultat

## Bruk

Når nye smoke eller testreferansar kjem, skal dei peike hit som basis.
