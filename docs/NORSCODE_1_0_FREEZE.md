# Norscode 1.0-konvergensfrys

Fram til `./bin/nc completion-gate --release` er grøn, er aktiv utvikling
avgrensa til feilretting, testar, portar, runtime/promotering, releasearbeid og
dokumentasjon som blir oppdatert frå faktiske gatebevis.

Nye AI-, web- og språkfunksjonar høyrer til etter Norscode 1.0. Endringar i
desse områda er berre tillatne når dei rettar ein eksisterande kontrakt eller
ein raud obligatorisk port.

`completion-gate --local` er lokal førehandskontroll. Berre
`completion-gate --release`, køyrd på rein commit med signerte
plattformattestasjonar, er globalt ferdigbevis.
