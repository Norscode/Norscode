# Selfhost Fase 3 - Regresjon

Dette dokumentet samlar dei første regresjonsreferansane for fase 3.

## Første regressionsmål

- AST-kontrakt skal halde stabil minimumsform
- Semantic-kjernen skal framleis avvise duplikat og halde scope-intensjon
- IR-løypa skal framleis vere dokumentert som normal bytecode-veg

## Kva som skal vaktast

- Endringar i `selfhost/ast.no`
- Endringar i `selfhost/semantic.no`
- Endringar i `selfhost/compiler/ir_to_bytecode.no`

## Bruk

Dette er referansepunktet for fase-3-regresjonar. Når nye smoke-testar kjem, skal dei peike tilbake hit.
