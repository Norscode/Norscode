# Norscode AI sikkerhet manuell innkobling v130

Denne fila er en MANUELL instruks.
Den endrer ikke ai.no automatisk.

Forutsetning:
- v125 import-prøve OK.
- v128 hovedfil-probe OK.
- v129 innkobling-klar OK.
- ai.no OK før endring.

Manuell innkobling:

1. Ta backup:
   cp ai.no ai.no.bak_sikkerhet_manuell_v130

2. Sjekk før endring:
   nc check ai.no
   nc check ai_moduler/sikkerhet.no

3. Vis bruk-linjer:
   grep -n '^bruk ' ai.no
   sed -n '1,60p' ai.no

4. Legg inn denne linjen manuelt sammen med andre bruk-linjer:
   bruk ai_moduler.sikkerhet som sikkerhet

5. Ikke endre noe annet.

6. Test:
   nc check ai.no

7. Hvis OK:
   printf "hjelp" > spm.txt
   nc run ai.no

8. Hvis feil:
   cp ai.no.bak_sikkerhet_manuell_v130 ai.no
   nc check ai.no

Regel:
Bare sikkerhet.no kobles inn nå. Ingen andre moduler.
