# Norscode AI kode_kalkulator manuell patch v303

Dette er bare en MANUELL patch-mal.
Ikke bruk automatisk.

Senere manuell metode:

1. Backup:
   cp ai.no ai.no.bak_kode_kalkulator_v303

2. Legg eventuelt inn import:
   bruk ai_moduler.kodegen_kalkulator_v301 som kodekalk

3. Endre bare kroppen til kode_kalkulator():

funksjon kode_kalkulator() -> tekst {
    returner kodekalk.kode_kalkulator()
}

4. Test:
   nc check ai.no
   printf "kod kalk.no = kalkulator" > spm.txt
   nc run ai.no

5. Hvis feil:
   cp ai.no.bak_kode_kalkulator_v303 ai.no
   nc check ai.no
