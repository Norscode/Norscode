# Norscode AI prosjektanalyse

Dette er ein trygg automatisk rapport frå ai_assistent/ai_prosjekt.no.
Assistenten les struktur og status, men endrar ingen viktige prosjektfiler.

v4 sjekkar kjende markeringsfiler.

## Viktige delar

[OK] compiler — kompilator og språk-kjerne
[OK] selfhost — Norscode som kan byggje seg sjølv
[OK] std — standardbibliotek
[OK] tests — testar
[OK] docs — dokumentasjon
[OK] frontend — frontend/web/UI
[OK] backend — backend/server/native
[OK] Hjemmeside — nettside
[OK] NorsDB — databaseprosjekt
[OK] vscode-norscode — VS Code extension
[OK] intelligence — AI/intelligens-modular
[OK] autonomous — autonome agentar
[OK] toolchain — bygg/verktøykjede
[OK] bin/nc — lokal Norscode-køyrar

## Trygg utviklingsplan

1. Analyser prosjektstruktur.
2. Lag rapport før endring.
3. Lag patch-fil, ikkje endre direkte.
4. Ta backup før endring.
5. Kjør nc check på endra filer.
6. Kjør relevante testar.
7. Skriv logg i ai_assistent/logg.txt.
