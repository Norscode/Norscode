# Random ID plan v663

Ekte session_id må:
- være tilfeldig
- være minst 128 bit
- ikke være brukerdata
- ikke inneholde rolle
- lagres kun som nøkkel til server-side session-store

Kartlegg neste:
grep -R "random\|uuid\|crypto\|bytes" -n prosjekter/Norscode/std prosjekter/Norscode/selfhost prosjekter/Norscode/compiler 2>/dev/null
