# Norscode webserver testplan v429

Før faktisk webserver bygges:
1. nc check prosjekter/NorscodeWeb/server_plan.no
2. nc run prosjekter/NorscodeWeb/server_plan.no
3. nc check prosjekter/NorscodeWeb/sider/index.no
4. nc check prosjekter/Norscode/tests/test_norscode_web_plan_v430.no

Senere når nc serve finnes:
1. nc serve prosjekter/NorscodeWeb/server.no
2. curl http://localhost:8080/
3. curl http://localhost:8080/public/style.css
4. curl http://localhost:8080/../../ai.no skal blokkeres
