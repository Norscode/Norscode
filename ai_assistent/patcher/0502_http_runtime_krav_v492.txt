# Norscode Web HTTP runtime krav v492

For ekte nc serve trengs:

1. TCP socket eller HTTP-server runtime
2. lese request line
3. hente path
4. kalle app.handter(path)
5. skrive HTTP response til klient
6. håndtere connection close
7. feilhåndtering og logging

Hvis Norscode ikke har socket/HTTP runtime ennå:
- lag bridge først
- ikke endre compiler/bin/nc direkte
