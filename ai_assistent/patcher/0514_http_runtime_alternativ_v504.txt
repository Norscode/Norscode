# Norscode HTTP runtime alternativ v504

Mulige veier:

Alternativ A: Native Norscode HTTP
- legge til std/http.no og std/socket.no senere
- krever runtime-støtte
- mest selvstendig

Alternativ B: nc serve i bin/nc
- ny kommando i CLI
- krever grundig test
- må ikke gjøres automatisk

Alternativ C: Midlertidig bridge
- ekstern liten HTTP-wrapper
- wrapper kaller Norscode app.no
- raskest å teste
- ikke endelig mål

Anbefaling:
Start med kartlegging, så bridge-prototype hvis native socket mangler.
