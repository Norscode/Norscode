# Norscode Web static-router deploy-plan v479

Når ekte server finnes senere:
- HTTP request path sendes til app_router.handter(sti)
- response-strengen sendes til klient
- static brukes for /public/*
- render brukes for side-ruter

Produksjon må ha:
- logging
- feilhåndtering
- timeout
- blokkering av farlige stier
- HTTPS via proxy eller innebygget TLS senere
