# Norscode bridge runtime plan v505

Hvis socket/http mangler:

Bridge-idé:
1. HTTP-wrapper lytter på port 8080.
2. For hver request henter wrapper path.
3. Wrapper kaller en Norscode-fil med path.
4. Norscode app.handter(path) returnerer response.
5. Wrapper sender response til nettleseren.

Fordel:
- Lar oss teste .no på server før native nc serve er ferdig.

Ulempe:
- Ikke 100% Norscode runtime ennå.
