# Norscode AI kode_ai_demo manuell patch v363

Dette er bare en MANUELL patch-mal.
Ikke bruk automatisk.

Senere manuell metode:

1. Backup:
   cp ai.no ai.no.bak_kode_ai_demo_v363

2. Legg eventuelt inn import:
   bruk ai_moduler.kodegen_ai_demo_v361 som kodeai

3. Endre bare kroppen til kode_ai_demo():

funksjon kode_ai_demo() -> tekst {
    returner kodeai.kode_ai_demo()
}

4. Test:
   nc check ai.no
   printf "kod ai_demo.no = ai" > spm.txt
   nc run ai.no

5. Hvis feil:
   cp ai.no.bak_kode_ai_demo_v363 ai.no
   nc check ai.no
