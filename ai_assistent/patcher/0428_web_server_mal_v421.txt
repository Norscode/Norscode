# Norscode Web Server mål v421

Mål:
- Bruke .no-filer på server.
- .no-filer skal kunne returnere HTML/JSON.
- Serveren skal route URL til riktig .no-fil.
- HTML/CSS kan fortsatt genereres, men kilden skal være Norscode.

Hovedflyt:
nettleser -> HTTP request -> Norscode router -> side.no -> HTML response
