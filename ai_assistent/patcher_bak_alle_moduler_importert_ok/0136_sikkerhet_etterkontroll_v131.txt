# Norscode AI sikkerhet etterkontroll v131

Denne fila endrer ikke ai.no.

Mål:
- Kontrollere systemet etter eventuell manuell innkobling av sikkerhet.no.
- Bekrefte at ai.no fortsatt er stabil.
- Bekrefte at bare sikkerhet-modulen er koblet inn, hvis manuell innkobling er utført.

Etterkontroll:

1. Sjekk ai.no:
   nc check ai.no

2. Sjekk sikkerhet-modul:
   nc check ai_moduler/sikkerhet.no

3. Sjekk test:
   nc check prosjekter/Norscode/tests/test_sikkerhet_etterkontroll_v131.no

4. Kjør AI:
   printf "hjelp" > spm.txt
   nc run ai.no

5. Kjør status:
   printf "status" > spm.txt
   nc run ai.no

Godkjent hvis:
- ai.no er OK
- sikkerhet.no er OK
- AI kjører uten parserfeil
