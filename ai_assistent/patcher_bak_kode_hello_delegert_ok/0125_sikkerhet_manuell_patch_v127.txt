# Norscode AI manuell sikkerhet-patch v127

Dette er kun en MANUELL patch-mal.
Denne pakken endrer ikke ai.no automatisk.

Mål:
- Koble inn ai_moduler/sikkerhet.no senere, men bare manuelt.
- Bruke importsyntaksen som ble testet OK i v125:
  bruk ai_moduler.sikkerhet som sikkerhet

Manuell patch senere:

1. Ta backup:
   cp ai.no ai.no.bak_sikkerhet_v127

2. Sjekk før endring:
   nc check ai.no
   nc check ai_moduler/sikkerhet.no
   nc run prosjekter/Norscode/examples/sikkerhet_import_probe_v125.no

3. Åpne øverst i ai.no:
   sed -n '1,80p' ai.no

4. Legg manuelt til sammen med andre bruk-linjer:
   bruk ai_moduler.sikkerhet som sikkerhet

5. Ikke endre noe annet.

6. Test etter endring:
   nc check ai.no

7. Hvis feil:
   cp ai.no.bak_sikkerhet_v127 ai.no
   nc check ai.no

Regel:
Kun én modul kobles inn om gangen.
