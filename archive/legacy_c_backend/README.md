# Legacy C-runtime for bootstrap

Denne mappa inneholder eldre C-runtime-kode som er flytta ut av aktiv vedlikeholdsflate:

- `nc_runtime_full.c`
- `nc_runtime_mini.c`
- `nc_native_main.c`

Filene skal **ikkje** vere del av normal byggkjede. Dei er her som historisk/vedlikeholdsreferanse dersom vi treng re-aktivering av C-bakenden i ei nødsituasjon.
