# Norscode AI samlet status

Denne fila samlar status frå AI-assistenten.

## Prosjekt

[OK] compiler -> compiler/compiler.no
[OK] std -> std/fil.no
[OK] tests -> tests
[OK] bin/nc -> bin/nc

## AI-assistent filer

[OK] rapport -> ai_assistent/rapport.txt
[OK] utviklingsplan -> ai_assistent/utviklingsplan.txt
[OK] testplan -> ai_assistent/testplan.txt
[OK] patch-forslag -> ai_assistent/patch_forslag.txt
[OK] logg -> ai_assistent/logg.txt
[OK] filrapport -> ai_assistent/filrapport.txt
[OK] patcher/0001 -> ai_assistent/patcher/0001_trygg_ai_leser_status.patch.txt

## Neste steg

1. Kjør ai_leser.no for å lese utvalde filer.
2. Kjør ai_patch.no for å lage patch-forslag.
3. Kjør ai_test.no før endringar.
4. Ikke endre compiler automatisk enno.
