Norscode AI Assistent v4

Trygg prosjektassistent for Norscode.

Filer:
- ai_prosjekt.no: sjekker prosjektstruktur
- ai_plan.no: lager utviklingsplan
- ai_test.no: lager testplan
- ai_patch.no: lager patch-forslag
- ai_logg.no: lager logg
- ai_leser.no: leser utvalgte filer trygt
- ai_status.no: lager samlet status
- patcher/: mappe for patch-forslag

Kjor fra prosjektroten:
nc run ai_assistent/ai_status.no
nc run ai_assistent/ai_leser.no
