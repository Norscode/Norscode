2. Køyr: printf "valider std patch Norscode fil" > spm.txt && nc run ai.no
3. Køyr: cd prosjekter/Norscode && nc check tests/test_std_fil_ai.no && nc run tests/test_std_fil_ai.no && cd ../..
4. Lag ny vurdering før std/fil.no blir rørt.
