# Norscode Roadmap

Norscode-roadmapen er no styrt av konkrete gates, ikkje lause fasetal.

## Hovudmål

- Gjer aktiv utvikling, bygg, release og CI sjølvstendig utan Python og C.
- Hald normal kjede kort: `.no` -> NCB JSON -> `selfhost/vm.no`.
- Løft kvart modenheitsområde til 10/10 berre når kodebevis, bruksbevis og gatebevis finst.

## Neste Praktiske Løft

- 9/10-golv: smoke/selftest for alle område, samla i `./bin/nc maintenance maturity`.
- 9/10-releasekandidat: dokumentert kontrakt, eksempel og gate for kvar brukarflate.
- 10/10-produktflate: installasjon, prosjekt, app, test, release og rollback kan gjennomførast utan intern prosjektkunnskap.
