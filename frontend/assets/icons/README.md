# Norscode Icons

Dette er den valde standardpakken for Norscode-logo og ikoner. Pakken brukar den innebygde N-logoen som felles kjelde for nettside, app-pakker, installasjonsskript og editorintegrasjonar utan Python- eller C-baserte genereringssteg i normal bruk.

## Filer

- `norscode-logo-dark.svg`: primærlogo for header-, dokumentasjons- og produktflater.
- `norscode-mark.svg`: kompakt app-/produktmerke basert på same N-logo.
- `favicon.svg`: nettleserikon.
- `file-main-no.svg`: primært `.no`-filikon.
- `file-utils-nc.svg`: `.nc`-filikon.
- `file-config-ncf.svg`: `.ncf`-filikon.
- `file-data-ncd.svg`: `.ncd`-filikon.
- `file-package-ncp.svg`: `.ncp`-filikon.
- `file-component-no.svg`: alternativt komponent-/source-ikon.
- `folder-project.svg`: prosjektmappe.
- `folder-package.svg`: pakkemappe.

## Bruk

Bruk `norscode-logo-dark.svg` som hovudlogo. Bruk `norscode-mark.svg` der logoen skal opptre som app-ikon, små nav-element og release-pakker.

Filtypeikon for macOS blir henta frå denne mappa av `tools/install-macos-file-icons.no`.
