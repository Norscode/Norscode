# Norscode Brand Assets

This folder contains the canonical Norscode logo and icon sources.

- `norscode-logo.svg` is based on the homepage dark logo
- `norscode-mark.svg` is the standalone app icon derived from the same mark
- `favicon.svg` is the small-size favicon source
- `file-*.svg` are macOS file type icon sources used by
  `tools/install-macos-file-icons.sh`
- `folder-*.svg` are project/package folder icon sources. macOS folder icons
  are applied per folder, so these are packaged as resources rather than
  globally registered by extension.

Rendered assets are stored alongside the SVG sources.
