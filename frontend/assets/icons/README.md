# Norscode Icons

Dette er den valde standardpakken for Norscode-logo og ikoner. Pakken er rein SVG og kan brukast av nettside, app-pakker, installasjonsskript og editorintegrasjonar utan Python- eller C-baserte genereringssteg.

## Filer

- `norscode-logo-dark.svg`: primærlogo for mørke flater og hero/header-bruk.
- `norscode-mark.svg`: kompakt app-/produktmerke.
- `favicon.svg`: nettleserikon.
- `file-main-no.svg`: primært `.no`-filikon.
- `file-utils-nc.svg`: `.nc`-filikon.
- `file-config-ncf.svg`: `.ncf`-filikon.
- `file-data-ncd.svg`: `.ncd`-filikon.
- `file-package-ncp.svg`: `.ncp`-filikon.
- `file-component-no.svg`: alternativt komponent-/source-ikon.
- `folder-project.svg`: prosjektmappe.
- `folder-package.svg`: pakkemappe.

## Bruk

Bruk `norscode-logo-dark.svg` som hovudlogo når flata er mørk eller kan ha mørk bakgrunn. Bruk `norscode-mark.svg` der ordmerket blir for breitt, til dømes app-ikon, små nav-element og release-pakker.

Filtypeikon for macOS blir henta frå denne mappa av `tools/install-macos-file-icons.sh`.
